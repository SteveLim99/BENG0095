# BENG0095

## 1. Setting up

- **Windows**

    1. Install virtualenv and setup virtualenv

        - `pip install virtualenv`
        - `virtualenv env`
        - `.\env\Scripts\activate`
    
    2. Install packages

        - `pip install -r requirements.txt`

- **Linux**

  1. Install virtualenv and setup virtualenv

     - `pip install virtualenv`
     - `virtualenv env`
     - `source env/bin/activate`

  2. Install packages
  
     - `pip install -r requirements.txt`
